##定义方法

&emsp;&emsp;方法使用标准函数格式、可访问性和可选的 `static` 修饰符来声明。例如：

```javascript
        class MyClass
        {
            public string GetString()
            {
                return "Here is a string."
            }
        }
```

>&emsp;&emsp;**与公共字段一样，.NET Framework 中的公共方法也采用 `PascalCasing` 形式来命名。**
